# Text-To-Math-Problem-Solver-Using-Google-Gemma-2

This project is a **Streamlit-based web application** that leverages **LangChain** and **Google Gemma 2** to solve mathematical problems and assist with data search. It provides users with an interactive chatbot interface to solve math problems, perform reasoning tasks, and fetch information from Wikipedia.

## Features

- **Mathematical Problem Solver**: Uses advanced language models to solve math problems with detailed, step-by-step explanations.
- **Reasoning Assistant**: Handles logic-based and reasoning questions.
- **Wikipedia Integration**: Fetches relevant information from Wikipedia for user queries.
- **Interactive Chat Interface**: Allows users to interact with the assistant in a conversational format.

## How It Works

1. **Input Your Question**: Enter your math or reasoning question in the text area.
2. **Choose Tools**: The app uses tools like a calculator, reasoning engine, and Wikipedia search to process your query.
3. **Get Results**: The assistant provides detailed answers and explanations.
